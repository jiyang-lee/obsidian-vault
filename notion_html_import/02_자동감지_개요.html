# 가장 위험한 3가지

1. 0.65/0.35가 최적처럼 보이지만 실제 sweep에서는 0.9/0.1이 더 좋음
2. 새 raw 데이터 재예측에 필요한 risk/leadtime/current-best 본체가 없음
3. canonical window에서 빠진 26개가 모두 pre_fault라 평가/커버리지 해석에 영향 가능
